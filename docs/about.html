The Muyumoqo Archaeological Project

Project Overview

Muyumoqo is an archaeological excavation project focused on analyzing lithic (stone tool) assemblages and obsidian artifacts. This research helps us understand:

  • How ancient peoples used and manufactured tools
  • Settlement patterns and site use over time
  • Trade networks through obsidian sourcing
  • Cultural practices through tool types and distributions

The Data

What We Collected

Our excavation generated data in several forms:

  1. Lithic Analysis Data - Detailed measurements and classifications of stone tools
  2. Obsidian Data - Specific information about obsidian artifacts from excavation units
  3. Contextual Information - Where artifacts were found, at what depth, in which excavation units

Data Files

  • lithic_analysis(analysis).csv - Stone tool measurements and attributes
  • obsidian_data_26(UNITS 8A 8B).csv - Obsidian artifacts from Units 8A and 8B
  • 2023_muyumoqo fichas - UE5.csv - Field documentation from Unit 5

Getting Started with R (For Beginners)

If you want to run this analysis yourself:

Step 1: Install R and RStudio

  • Download R from https://www.r-project.org/
  • Download RStudio (the free version) from https://posit.co/download/rstudio-desktop/

Step 2: Clone or Download This Project

Download the entire project folder to your computer.

Step 3: Open the Project

Open Archaeology.Rproj in RStudio. This sets up your working environment.

Step 4: Restore the Packages

In the RStudio console, type:

renv::restore()

This installs all the R packages needed, with the exact versions we used.

Step 5: Open the Analysis Files

In the analysis/ folder, open any .Rmd file and click “Knit” to run the analysis!
